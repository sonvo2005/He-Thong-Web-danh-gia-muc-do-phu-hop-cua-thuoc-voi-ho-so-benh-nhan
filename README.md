# He Thong Web Danh Gia Muc Do Phu Hop Cua Thuoc Voi Ho So Benh Nhan

Ung dung ASP.NET Core MVC ho tro tiep nhan ho so benh nhan, quan ly dac tinh thuoc va hien thi ket qua danh gia muc do phu hop cua thuoc.

## Chay du an

```powershell
dotnet restore WebDanhGiaHs\WebDanhGiaHs.csproj
dotnet run --project WebDanhGiaHs\WebDanhGiaHs.csproj
```

Connection string nam trong `WebDanhGiaHs/appsettings.json`.
